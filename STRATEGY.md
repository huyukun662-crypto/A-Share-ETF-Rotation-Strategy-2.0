## Definition

```
# ──────────────────────────────────────────────────────────────────
# Step 1: LEG A — 惩罚式单 ETF 打分(抓"稳涨未爆量"的 ETF)
# ──────────────────────────────────────────────────────────────────
mom_4w[i,t]  = cum[i,t] / cum[i,t-4] - 1                    # 4 周收益
turn_4w[i,t] = mean(turnover_mv[i, t-3..t])                 # 4 周平均换手
breadth[t]   = z_{52w}(%全市场 above-MA20)                   # 市场广度 z

score_A[i,t] = z_{cs}(mom_4w[i,t])
             - 1.5 * z_{cs}(turn_4w[i,t])                    # 拥挤罚分
             + 0.3 * breadth[t]

w_A[t] = 等权 top-4 by score_A                               # 25% 每只

# ──────────────────────────────────────────────────────────────────
# Step 2: LEG G — 9 组 super-industry 轮动(抓主线领头羊)
# ──────────────────────────────────────────────────────────────────
# 9 组: 科技成长/新能源/高端制造/大金融/大消费/周期资源/地产链/农业/红利
group_mom_4w[g,t] = 组内等权平均 4w 收益
top_3_groups[t]   = argmax(group_mom_4w)[:3]

for g in top_3_groups:
    leader = argmax(mom_4w[i,t] for i in group_g and eligible)
    pick leader

w_G[t] = 等权 top-3 leaders(33.3% 每只,最多 3 只 ETF)

# ──────────────────────────────────────────────────────────────────
# Step 3: ENSEMBLE 两腿 50/50 合并
# ──────────────────────────────────────────────────────────────────
w_raw[i,t] = 0.5 * w_A[i,t] + 0.5 * w_G[i,t]   # 7 只 ETF,合计 100%

# ──────────────────────────────────────────────────────────────────
# Step 4: 市场 Gate(熊市切避险)
# ──────────────────────────────────────────────────────────────────
mkt_cum[t]  = 全 34 ETF 等权累计指数
gate_on[t]  = mkt_cum[t] > mean(mkt_cum[t-50w..t])          # 50 周 MA

if gate_on[t]:
    w_pre_vt[t] = w_raw[t]
else:
    w_pre_vt[t] = {159934.SZ: 1.0}                         # 100% 黄金 fallback

# ──────────────────────────────────────────────────────────────────
# Step 5: Vol Target 15% 年化(只缩不放)
# ──────────────────────────────────────────────────────────────────
rolling_vol_26w[t] = 组合过去 26 周已实现年化波动
scale[t]           = min(1.0, 0.15 / rolling_vol_26w[t])
w_final[t]         = w_pre_vt[t] * scale[t]

# ──────────────────────────────────────────────────────────────────
# 执行
# ──────────────────────────────────────────────────────────────────
# - 周五收盘计算信号,下周一开盘执行(delay = 1)
# - 每周再平衡
# - 成本:5 bps / side on turnover
# - Universe: 34 A 股主题 ETF(staggered join: 上市 12w 后进入池)
```

## 为什么有效

**3 个互补机制**,每个都能在回测里独立证伪:

1. **"动量 + 拥挤惩罚"双轴**(Leg A)—— 过热赛道罚分,"涨得稳"的 ETF 反而加分;这避开了 2021 白酒 / 2023 CPO 式的熄火崩塌。
2. **9 组主线识别**(Leg G)—— 单 ETF 层面噪声大,板块层面更稳;group rotator 提供结构性 alpha 分散。
3. **黄金 fallback 是真 anti-cyclic**(2022 A 股 -22% 熊市,黄金 +9.2%;2024 美联储转向时金价 +23%);用户提的红利/铜/港股/纳指在 2022 全球同步下跌里都是 long-beta,只有黄金真避险。

## 风险与局限(诚实披露)

1. **2020 COVID DD 无法消除**。MA50 慢 gate 对 4 周急跌 V 型反应过慢,Full MaxDD 始终卡在 -8.89%。要压下去需切日频 gate 或压低 vol target(都会牺牲 Sharpe)。
2. **黄金 beta 依赖**。OOS 2024-2026 Sharpe 2.0+ 有相当部分来自金价 2024-2025 +23% 上涨;若金价回归横盘,长期 Sharpe 预期回落到 **1.4-1.7**。
3. **策略是 long-beta 跟随增强**,不是 market-neutral。牛市表现好,熊市靠 gate + 黄金 + vol target 三重防御,但**不会 short**。
4. **样本不含 2018 深熊**。2018 A 股 -25% 系统性熊市下策略表现未知(ETF 覆盖不够);推测 gate 会触发,但黄金 2018 仅小涨 +3%,防御有限。
5. **快 gate 缺席**。无法抓 COVID-type 突发急跌;如要补,参见 V7_gb7030_FG4 变体。
6. **黄金 ETF 的跟踪误差**。回测用 159934.SZ 实际净值,已含跟踪误差;但若 ETF 停牌或折溢价异常,需人工干预。
7. **34 主题 ETF 样本短**。部分 ETF 仅 2021+ 上市,staggered join 下早期横截面只有 6-10 只可 rank;对早期 2019-2020 数据敏感。
